Demo 03: Orchestrator-first (Kontrast)
======================================
Kontrast zu Demo 01/02 | Manueller Ablauf

WAS WIR ZEIGEN: Der Mensch (Orchestrator) steuert jeden MCP-Aufruf bewusst.
Das LLM wird erst am Ende fuer die Zusammenfassung genutzt.

KONTRAST ZU DEMO 01/02:
- LLM-first:        LLM entscheidet, welche Tools es aufruft
- Orchestrator-first: Mensch/Code entscheidet jeden Schritt

WANN ORCHESTRATOR-FIRST?
- Compliance: Jeder Schritt muss nachvollziehbar sein
- Kosten: LLM-Aufrufe minimieren
- Kontrolle: Definierter Ablauf ohne Ueberraschungen

VERFUEGBARE MCP-BAUSTEINE:
- Tool: manual.docs.search (keyword)
- Resources: manual/docs/catalog, manual/docs/document/{id}
- Dokumente: manual-coffee, manual-overview, manual-standup

VORBEREITUNG
------------
Terminal 1: cd 03/server && dotnet run
Terminal 2: cd 03/client && dotnet run

Server laeuft auf http://localhost:5000/sse

SCHRITT-FUER-SCHRITT (MENUE-BASIERT)
------------------------------------

1. KATALOG ANZEIGEN
   Taste: 1 (Liste Katalog)

   -> Erklaere: WIR entscheiden, wann der Katalog gelesen wird
   -> Zeige: Dokument-IDs fuer naechste Schritte

2. SUCHE AUSFUEHREN
   Taste: 2 (Suche per Tool)
   Eingabe: coffee

   -> Erklaere: WIR geben den Suchbegriff vor, nicht das LLM
   -> Zeige: Suchergebnisse mit IDs

3. ERSTES DOKUMENT LESEN
   Taste: 3 (Dokument lesen)
   Eingabe: manual-coffee

   -> Erklaere: Dokument wird in Zwischenspeicher gelegt
   -> Zeige: Text erscheint, kein LLM-Aufruf bisher

4. ZWEITES DOKUMENT LESEN
   Taste: 3 (Dokument lesen)
   Eingabe: manual-standup

   -> Erklaere: Wir sammeln mehrere Dokumente manuell

5. ZWISCHENSPEICHER PRUEFEN
   Taste: 4 (Gesammelte Dokumente anzeigen)

   -> Erklaere: Alles was bisher passiert ist, war OHNE LLM
   -> Zeige: Liste der gesammelten Dokumente

6. LLM EINSETZEN
   Taste: 5 (Zusammenfassung per LLM)

   -> Erklaere: JETZT ERST rufen wir das LLM auf
   -> Zeige: LLM fasst die manuell gesammelten Dokumente zusammen

KERNAUSSAGE
-----------
"In Demo 01/02 hat das LLM entschieden, was es liest.
 Hier haben WIR entschieden - und das LLM nur am Ende genutzt.
 Beide Ansaetze haben ihre Berechtigung."
